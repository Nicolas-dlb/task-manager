## Getting Started

To run the project follow these simple steps.

### Installation

1. Clone the repo

```sh
git clone https://github.com/Nicolas-dlb/task-manager
```

2. install dependances

```sh
npm install
```

3. run the project

```sh
npm start
```

### Tests

run unit tests

```sh
npm test
```
